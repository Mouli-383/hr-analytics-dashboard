# HR Analytics Dashboard

## Project Overview

The HR Analytics Dashboard is an interactive Tableau project designed to analyze employee demographics, termination trends, salary distribution, workforce composition, and organizational performance. The dashboard enables users to monitor key HR metrics and gain insights into workforce trends through interactive visualizations and filters.

---

## Business Problem

Human Resources teams require visibility into workforce trends to support hiring, retention, compensation planning, and performance management. However, large employee datasets can make it difficult to identify patterns and monitor key workforce metrics.

This dashboard consolidates HR data into an interactive analytical solution, enabling users to explore employee demographics, workforce distribution, salary trends, and employee termination patterns.
